# Portfolio Monorepo

Monorepo con instancias separadas de frontend y backend.

## Estructura

```
portfolio/
├── frontend/        # Angular - Arquitectura Hexagonal
└── backend/         # NestJS + PostgreSQL - Arquitectura Hexagonal
```

## Requisitos

- Node.js >= 20
- npm >= 10
- PostgreSQL >= 15

## Instalación

```bash
npm run install:all
```

## Desarrollo

```bash
# Frontend
npm run frontend

# Backend
npm run backend
```
